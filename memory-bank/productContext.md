# Product Context

## Target Users
- Visitors to developer portfolio site
- Recruiters exploring candidate background
- Potential collaborators seeking project details

## Problem Statement
Static portfolio sites lack interactivity. Visitors must navigate multiple pages to find specific information about projects, skills, or experience.

## Solution Approach
Conversational AI interface that:
- Understands user intent (asking about project, profile, skills)
- Retrieves relevant context
- Generates natural language responses via LLM

## User Experience Goals
- Fast response time
- Accurate, relevant answers
- Natural conversational flow
- Graceful handling of off-topic queries
